## เกิดอะไรขึ้น

URL ที่คุณเปิดไม่มีอยู่ใน docs site ของ CUVETSMO. สาเหตุที่น่าจะเป็นไปได้:

- ลิงก์เก่าจากก่อนปรับโครงสร้างเอกสาร
- พิมพ์ URL ผิด
- หน้านี้ถูกย้ายหรือเปลี่ยนชื่อ

## ลองไปทางนี้แทน

- [Overview](/) — index ของ docs ทั้งหมด
- [Architecture](/architecture/) — system reference, data model, edge functions
- [SSO Design](/architecture-sso/) — cross-app identity draft
- [Developer Onboarding](/dev-onboard/) — clone, run, ship first PR
- [Successor Guide](/successor/) — annual handover playbook

หรือใช้ปุ่มค้นหามุมขวาบน — กด `Ctrl + K` (Windows) / `Cmd + K` (Mac) ก็ได้
