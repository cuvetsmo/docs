## ทำไมต้องมี docs site นี้

WebCUVETSMO เป็น web app ของสโมสรนิสิตคณะสัตวแพทยศาสตร์ จุฬาฯ — มี approval workflow · AI chat · ชมรม · event calendar · marketplace · และเป็นโครงสร้างที่ตั้งใจ **ส่งต่อให้คนรุ่นถัดไปดูแลได้จริง** ไม่ใช่แค่ Palm คนเดียว

เอกสารทั้ง 4 หน้าด้านล่างคือสิ่งที่ Vet 87 / 88 / 89 (หรือ open-source contributor คนใดก็ตาม) ควรอ่านก่อนแตะ codebase

## Where these docs come from

ทุกหน้าใน docs site นี้ถูก source มาจาก folder [`docs/` ของ WebCUVETSMO](https://github.com/palmzamak2547/WebCUVETSMO/tree/main/docs) — เป็น docs ที่ commit อยู่กับ codebase จริง ไม่ใช่เอกสารแยกที่หลุด sync ได้ง่าย
